# Blueshield-Crime-Management-Community-Safety-App

## OVERVIEW

BlueShield is a mobile application designed to streamline crime tracking, optimize officer deployment, and improve communication between law enforcement and the community. Features include real-time incident mapping, automated officer dispatch, crime data analytics, community safety reporting, and direct citizen-officer engagement.

## WHAT IT DOES

### CRIME ANALYTICS DASHBOARD 
- Interactive graphs and charts to analyze crime trends over adjustable timeframes

### AUTOMATED OFFICER DEPLOYMENT  
- Real-time dispatch based on officer proximity, availability, and incident location

### CRIME DATA HUB
- Centralized database for past, current, and closed cases with search/filter capabilities

### COMMUNITY SAFETY TRACKER 
- Residents report concerns with geotagging and prioritization

### LIVE OFFICER MAP
- Real-time tracking of officer locations for efficient deployment

### INSTANT GROUP CHAT
- Direct communication between dispatched officers and command center

### CRIME AWARENESS NOTIFICATIONS
- Push alerts to public about ongoing threats and prevention tips

## COMMUNITY ENGAGEMNET HUB
- Q&A forums, resource sharing, and direct citizen-officer interaction

## KEY FINDINGS

### USABILITY EVALUATION RESULTS 

#### COGNITIVE WALKTHROUGH
- Key Issues Found: No direct officer chat, no emergency call button, limited search, no download feedback
- Resolutions: Added chat feature, emergency call button, general search, success/error popups

#### HEURISTIC EVALUATION (NIELSEN'S 10)
- Key Issues Found: Missing error messages for download failures
- Resolutions: Implemented confirmation popups

## DESIGN & EVALUATION METHODS 

- Research: Personas (Police Lieutenant, Crime Analyst, Community Liaison Officer), Scenario-based analysis
- Low-Fidelity: Storyboard, Wireframes
- High-Fidelity: Figma prototype with interactive transitions
- Usability Testing: Cognitive Walkthrough (user perspective)
- Expert Evaluation: Heuristic Evaluation (Nielsen's 10 principles)

## KEY IMPROVEMENTS AFTER EVALUATION 

- Issue: No direct officer communication
  - Solution: Added dedicated chat feature on community page & homepage

- Issue: No emergency call within app
  - Solution: Added emergency call button on bottom taskbar

- Issue: Limited search functionality
  - Solution: Added general search feature across entire app

- No download confirmation
  - Solution: Added success/error popup messages for report downloads
